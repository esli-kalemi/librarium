# Librarium

Librarium is a static library management dashboard and landing page built with HTML, CSS, and JavaScript. It includes a main marketing homepage and an admin dashboard for managing books, authors, categories, clients, and orders.

## Features

- Landing page with hero, features, articles, statistics, and About Us sections
- Smooth scrolling navigation links
- Admin dashboard with sidebar navigation
- Book management form for adding books
- Author, category, client, and order tables with inline row creation
- Report section with monthly sales and category sales visuals
- Responsive layout for modern library and bookstore use cases

## Files

- `index.html` - Landing page with homepage navigation and About Us video
- `dashboard.html` - Admin dashboard interface
- `style.css` - Landing page styles
- `dashboardstyle.css` - Dashboard page styles
- `dashboard.js` - Dashboard interactivity and table behavior
- `images/` - Image assets used throughout the site
- `videos/` - Demo video referenced from the About Us section

## Usage

1. Open `index.html` in a browser to view the homepage.
2. Open `dashboard.html` in a browser to use the admin dashboard.
3. For best results, serve the folder through a local web server if browser restrictions block media or script loading.

## Development

If you want to extend the project:

- Add new form validation and persistence
- Replace placeholder tables with real data sources
- Improve styling and responsive behavior
- Convert the site to a single-page application or backend-driven dashboard

## Notes

- This is a static frontend demo and does not include a backend or database.
